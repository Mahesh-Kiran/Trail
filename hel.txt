🚀 AMAZON LEX – HOTEL BOOKING BOT (DETAILED STEPS)
🔹 Step 1: Open Amazon Lex
Go to Amazon Web Services Console
Search Amazon Lex
Click Create bot
🔹 Step 2: Create Bot

Choose:

✔ Create a blank bot
Enter:
Bot name → HotelBookingBot

IAM Role:

✔ Create a new role
Keep defaults → Click Next
Select:
Language → English
Voice → optional
Click Done
🔹 Step 3: Create Intent
Go to Intents
Click Add intent → Create intent

Name:

BookHotel
Click Create
🔹 Step 4: Add Sample Utterances

Add user inputs:

I want to book a hotel
Book a room
Reserve hotel
I need a room

👉 These are Utterances (user sentences)

🔹 Step 5: Add Slot – Age (with Condition)
Click Add slot
Enter:
Slot name → age
Slot type → AMAZON.Number
Prompt → What is your age?

Mark:

✔ Required
🔥 Add Condition (IMPORTANT)

Go to:

Slots → Advanced options

Scroll:

Slot capture → success response → Conditional branching
Click Add condition
Add:
Condition: {age} < 18
Response: You are not eligible for hotel booking
🔹 Step 6: Add Slot – Location
Click Add slot
Enter:
Slot name → location
Slot type → AMAZON.City
Prompt → Which city do you want?
Mark Required
🔹 Step 7: Add Slot – Check-in Date
Click Add slot
Enter:
Slot name → checkin
Slot type → AMAZON.Date
Prompt → What is your check-in date?
Mark Required
Optional:
Retry prompt: Please provide valid date (e.g., 2026-03-25)
🔹 Step 8: Add Slot – Nights
Click Add slot
Enter:
Slot name → nights
Slot type → AMAZON.Number
Prompt → How many nights will you stay?
Mark Required
🔹 Step 9: Create Custom Slot Type (RoomType)
Go to Slot Types

Click:

Add slot type → Create blank

Name:

RoomType
Add values:
Single
Double
Suite
Save
🔹 Use Custom Slot in Intent
Go back to Intent
Add slot:
Slot name → roomType
Slot type → RoomType
🔹 Step 10: Add Response Cards (Buttons)
Go to slot prompt

Click:

More prompt options → Add card group
Add:
Title → Select Room Type

Buttons:

Single → "Single"
Double → "Double"
Suite → "Suite"
🔹 Step 11: Configure Responses
Initial Response:
Welcome to Hotel Booking! What is your name?
Confirmation Prompt:
Do you want to confirm booking in {location} for {nights} nights?

Click Build
Wait for success
